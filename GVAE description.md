**GVAE description**

The Scene Graph VAE is a standalone module whose job is to compress a structured 3D outdoor scene graph into two dense spatial volumes of latent features — one at a coarse resolution matched to the first level of the diffusion model, and one at a medium resolution matched to the second level. These volumes are KL-regularised toward a standard Gaussian, so the diffusion model can treat them as a layout prior and condition on them directly.

The input is a scene graph where every node carries a three-dimensional centroid position, a set of footprint semi-axes describing the node's volumetric extent, a one-hot semantic label, and a node type identifier. Edges encode physical proximity and road connectivity.

The architecture processes this graph through a three-level sequential encoding chain. At the finest level, the instance graph passes through a shallow two-to-three-layer relational graph attention network. The attention inside this network uses PointROPE — a parameter-free rotary positional embedding that divides the feature dimension into three equal parts corresponding to the x, y, and z axes, and applies standard one-dimensional rotary embeddings independently along each axis. Because rotary embeddings encode relative positions implicitly through the query-key dot product, every attention operation is automatically aware of the metric three-dimensional geometry between nodes, without any learned positional parameters.

The first coarsening step then produces region-level supernodes using farthest point sampling on the three-dimensional node positions, which guarantees spatial contiguity by construction — nearby instances are always grouped together, never spatially distant ones. Each supernode inherits its centroid as the weighted average of its members' positions, its footprint as the bounding box of its cluster, and its semantic distribution as the soft average of its members' labels. Critically, the embedding passed to the next level is not the raw supernode attributes — it is the pooled output of the instance-level graph attention network, which already encodes local relational context. The region-level encoder is initialised from a concatenation of these inherited embeddings with the supernode's own geometric attributes, projected to the working feature dimension. It then runs another shallow relational graph attention network with PointROPE, refining these initialised embeddings through message passing over the coarser graph topology.

A second coarsening step repeats the same procedure, this time producing scene-level supernodes — a handful of large spatial regions covering the entire scene. At this level, because the node count is small, a full exact multi-head attention is applied over all supernodes simultaneously, again with PointROPE. This is the only level where full-graph communication is both necessary and affordable. At finer levels, the local relational graph attention with its fixed receptive field is sufficient, because the coarsening operation automatically expands the physical extent covered by a fixed number of hops.

Each of the two coarser levels — region and scene — independently produces a dense three-dimensional latent volume. This is done through anisotropic Gaussian splatting: each node scatters its embedding across the target voxel grid using a kernel whose bandwidth is set directly from the node's footprint semi-axes. A building with a large footprint spreads its embedding over many voxels; a small vehicle covers just a few. The kernel is truncated at twice the standard deviation per axis for efficiency, and the accumulation is done via sparse scatter-add operations. The resulting splatted volume is then processed by a three-dimensional U-Net, which serves two purposes: it propagates information from node-occupied cells into empty background cells, and it smooths the splatted features into a continuous coherent field. The U-Net is shallower at the coarse level, where large-footprint supernodes already provide dense coverage, and deeper at the region level, where smaller footprints leave more empty space to inpaint. A final convolutional head produces the mean and log-variance of the voxel-wise Gaussian posterior, and a reparameterised latent sample is drawn.

The decoder exists solely to provide a reconstruction training signal. For each node, it computes a readout from the latent volume using deformable cross-attention. Twenty-seven reference points are initialised on a three-by-three-by-three grid within the node's footprint bounding box, then shifted by learned offsets derived from the node's embedding. The attention aggregates the latent features at these reference points. This is the structural inverse of the splatting operation: where splatting distributed a node's embedding outward to voxels weighted by a Gaussian kernel, the decoder aggregates voxel features back to the node weighted by a learned attention over a bounded neighbourhood. The aggregated representation feeds three small MLP heads that predict the node's semantic distribution, its centroid position, and its footprint. For proximity edges, reconstruction is framed as a position-consistency loss rather than binary edge prediction, avoiding the severe class imbalance problem that arises when most node pairs are not connected.

The occupancy loss provides geometric grounding. For a set of sampled three-dimensional query points, the decoder interpolates the latent volume and predicts binary occupancy. The ground truth comes from the actual three-dimensional scene, voxelised at the target resolution — not from a footprint proxy. Half the query points are sampled inside occupied voxels, half outside, to keep the signal balanced.

The training proceeds in four stages to manage the complexity of the joint loss. In the first stage, only the coarsening modules are trained, ensuring spatially coherent supernodes before any gradient flows through the embedding inheritance chain. In the second stage, the instance-level graph attention network, the first coarsening step, the region-level encoder, the medium-resolution splatting pipeline, and its decoder are all activated. In the third stage, the second coarsening, scene-level encoder, and coarse pipeline are added. In the fourth and final stage, the entire network is jointly fine-tuned at a reduced learning rate. The KL regularisation weight follows a cyclical annealing schedule throughout — it starts at zero and ramps up and then holds, repeatedly, to prevent posterior collapse while maintaining a useful variational bottleneck.

Once trained, the GVAE is completely independent of the diffusion model. At inference, the encoder chain runs on the input scene graph and produces the two latent volumes directly. These are consumed by the first and second levels of the hierarchical diffusion model as layout conditioning signals. All finer diffusion levels are conditioned exclusively on the cascade of coarser diffusion outputs — the scene graph plays no role there. The graph controls the general spatial layout, and then steps aside.
